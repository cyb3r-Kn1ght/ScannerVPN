# IP Pool API hướng dẫn sử dụng

1. Thêm nhiều target vào IP pool:

```sh
curl -X POST http://10.102.199.42:8000/api/ip_pool/add \
     -H "Content-Type: application/json" \
     -d '{"targets": ["scanme.nmap.org", "example.com", "192.168.1.1"]}'
```

2. Lấy danh sách target trong IP pool:

```sh
curl http://10.102.199.42:8000/api/ip_pool/list
```

3. Xoá target khỏi IP pool:

```sh
curl -X DELETE "http://10.102.199.42:8000/api/ip_pool/delete?target=scanme.nmap.org"
```
